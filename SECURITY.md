# Security Policy

## Supported Versions

We take security seriously. Below is the list of versions that are currently being supported with security updates. We highly recommend using the latest version of the plugin.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.x.x   | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerability in **LimitlessEnchants** (for example, a way for players to crash the server, duplicate items, or bypass permissions in an unintended way), please do **not** report it through public GitHub Issues.

Instead, please report it privately to ensure we have time to fix it before it becomes publicly known.

**How to report:**
1. Send a private message on Discord or email the repository owner directly.
2. Provide a detailed description of the issue.
3. Include steps to reproduce the vulnerability.

We will try to acknowledge your report within 48 hours and release a patch as quickly as possible.

Thank you for helping to keep our community safe!
